[45. Jump Game II](https://leetcode.com/problems/jump-game-ii/)
# 제한사항
- `1 <= nums.length <= 10^4`
- `0 <= nums[i] <= 100`
- `nums[n - 1]까지 도착 보장`

# 문제 정리
nums[n - 1]까지 점프를 통해서 도착한다.
가장 적은 횟수를 점프해서 도착할 때 점프 횟수는?
# 접근 방법
1. BFS
# 복잡도
- `n = nums.length`
- `m = nums[i]`
- `시간복잡도 : O(n*m)`

# 코드
``` cpp
class Solution {  
public:  
    int jump(vector<int>& nums) {  
        // 2. BFS  
        if (nums.size() == 1) {  
            return 0;  
        }  
        int destination = nums.size() - 1;  
        bool visited[nums.size()];  
        for (int i = 0; i < nums.size(); i++) {  
            visited[i] = false;  
        }  
  
        queue<int> queue;  
        for (int i = 1; i <= nums[0]; i++) {  
            queue.push(i);  
        }  
  
        int count_of_jump = 0;  
        while (!queue.empty()) {  
            int size_of_queue = queue.size();  
            for (int i = 0; i < size_of_queue; i++) {  
                int top = queue.front();  
                queue.pop();  
  
                if (top >= destination) {  
                    return count_of_jump + 1;  
                }  
                if (visited[top]) {  
                    continue;  
                }  
                visited[top] = true;  
  
                for (int j = 1; j <= nums[top]; j++) {  
                    queue.push(top + j);  
                }  
            }  
            count_of_jump++;  
        }  
  
        return 0;  
    }  
};
```
