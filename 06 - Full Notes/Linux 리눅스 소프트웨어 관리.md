[[Linux]] [[Linux CLI]]
# 검색

```Bash
apt-cache search [키워드]
```

# 설치

```Bash
apt-get install packagename
```

  

# 삭제

```Bash
apt-get remove [패키지명]
```

## 전체 삭제

```Bash
apt-get purge [패키지명]
```

  

# 업데이트

패키지 목록 업데이트

```Bash
apt-get update
```

# 버전 업그레이드

```Bash
apt-get upgrade
```
