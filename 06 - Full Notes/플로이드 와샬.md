#Algorithm #Graph #Notion 
[[Graph]] [[Algorithm Notion]]
1. 모든 정점과 정점 사이 거리
2. 음의 가중치를 가져도 괜찮다
3. 시간복잡도 : O(V^3)

``` cpp
for(int k = 1; k<= n; k++){
    for(int i = 1; i <= n; i++){
        for(int j = 1; j<=n; j++){
            dist[i][j] = min(dist[i][j], dist[i][k]+dist[k][j]);
        }
    }
}
```
